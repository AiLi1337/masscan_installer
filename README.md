# masscan_installer
masscan一键脚本
```bash
bash <(curl -sL https://raw.githubusercontent.com/AiLi1337/masscan_installer/main/masscan_installer.sh)
```
```bash
=================================================
            masscan 一键脚本 (v3.2)            
=================================================
 工作目录: /opt/masscan
-------------------------------------------------
 1. 安装 masscan
 2. 执行扫描 (自动清理旧日志)
 3. 查看扫描进程
 4. 查看扫描结果 (.xml 文件)  <- 新功能
 5. 手动删除 nohup.out 日志
 6. 卸载 masscan
 7. 退出脚本
=================================================
```
